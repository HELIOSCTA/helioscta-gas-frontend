# Frontend Styling Standards

Index of frontend standards. Each file is the single source of truth for its
topic — keep them focused and cross-link rather than duplicating content.

## Contents

- [`theme.md`](./theme.md) — palette, typography, color-accent rules.
- [`cash-hub-views.md`](./cash-hub-views.md) — Historicals section headers and
  tab/toggle consistency.
- [`ux-constraints.md`](./ux-constraints.md) — data-logic and responsive
  preservation guarantees.
- [`plotting.md`](./plotting.md) — Recharts conventions: palettes, axes,
  tooltips, legends, focus mode, sizing.
- [`parquet-pages.md`](./parquet-pages.md) — required `ParquetMetaStrip` in the
  page header for any parquet-backed page, dataset registry, API endpoint.
- [`mobile-experience.md`](./mobile-experience.md) — breakpoints, sidebar
  drawer, tap targets, full-detail tables (non-negotiable), viewport meta.
